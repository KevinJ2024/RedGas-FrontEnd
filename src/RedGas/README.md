# RedGas Project

This project is a web application built using React and Vite. It features a modern design with various sections including a header, hero, product categories, offers, heaters, and animations. The application utilizes Tailwind CSS for styling and Swiper for creating a carousel of cards.

## Project Structure

- **src/**: Contains all the source code for the application.
  - **App.jsx**: Main component that renders various layout components.
  - **index.css**: Global styles and utility classes.
  - **main.jsx**: Entry point of the application.
  - **Animations/**: Contains animation components.
    - **Animations.jsx**: Combines multiple animation components.
    - **AnimatedDots/**: Animated dots effect using canvas.
    - **ColorCircles/**: Animated color circles.
  - **assets/**: Contains image and SVG assets.
  - **Layouts/**: Contains layout components for different sections.
  - **UI/**: Contains reusable UI components such as cards, headers, and links.

## Installation

To install the necessary dependencies, run the following command:

```
npm install
```

## Scripts

- **dev**: Starts the development server.
- **build**: Builds the application for production.
- **lint**: Lints the code using ESLint.
- **preview**: Previews the production build.

## Usage

After installing the dependencies, you can start the development server with:

```
npm run dev
```

Open your browser and navigate to `http://localhost:3000` to view the application.
